# Project Management System

A comprehensive project management system consisting of a Java Spring Boot backend and a frontend application.

## Project Structure

- **[backend](file:///c:/Users/PC/Desktop/PROGRAMMING/Java/projectmanagementsystem/backend)** - The backend API application built with Java and Spring Boot.
- **[frontend](file:///c:/Users/PC/Desktop/PROGRAMMING/Java/projectmanagementsystem/frontend)** - The client-side single page application.
- **migrations** - Database migration scripts.

## Getting Started

### Backend
1. Navigate to the `backend` directory.
2. Run the application using Maven:
   ```bash
   ./mvnw spring-boot:run
   ```

### Frontend
1. Navigate to the `frontend` directory.
2. Install the dependencies and run the local development server:
   ```bash
   npm install
   npm run dev
   ```
